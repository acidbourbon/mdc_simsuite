&gas
gas-id "ar/co2 90/10"
global gas_file `gas/ar_co290-10-293-760.dat`
temperature 293
pressure 760
Call inquire_file(gas_file,exist)
If exist Then
Say "Gas file exists, retrieving ..."
get {gas_file}
Else
Say "Gas file not found, generating ..."
magboltz argon 90. carbon-dioxide 10. mobility 1.000000
write dataset "gas/ar_co290-10-293-760.dat" gasdata remark "magboltz-ar90-co210"
Endif
heed argon 90. carbon-dioxide 10.
